This is a fork of the OpenIGTLink-Java repository of WPI-AIM (https://github.com/WPI-AIM/OpenIGTLink-Java)
It contains a modernized, streamlined structure and adapted compatibility to newer Java Versions (at least Java 21).
This fork is maintained by NAMI-THU.


--------

This project provides a java implementation of the OpenIGTLink protocol defined in

          http://wiki.na-mic.org/Wiki/index.php/OpenIGTLink
